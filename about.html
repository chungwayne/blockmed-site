What is BlockMed

BlockMed proposes an open protocol which allows medical IoT data largely untethered to current legacy IT systems, to be rapidly adopted by payers, providers, and patients in a BMD token eco-system with core innovations in enhanced cybersecurity, crowdsourced AI algorithms and machine to machine payment.
