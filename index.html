BLOCKMED

HIPAA-compliant medical IoT data, on and off the blockchain — verified with the patient, provider, and payer.

Enhanced Cybersecurity

BlockMed proposes an open protocol which allows remote medical IoT data to be HIPAA compliant on and off the blockchain so there is record of sensor data verified with the patient, provider and payer.

Crowdsourced AI

Remotely generated medical IoT data can be crowdsourced to the BMD community to develop superior algorithms, that overcomes overfitting bias based on static historical data as opposed to continuously generated live data.

Machine to Machine Payments

Remote medical IoT data can result in real-time machine to machine payments through BMD token. Such a protocol has potential to transform transmission of value throughout the entire payer, pharma, patient ecosystem.
